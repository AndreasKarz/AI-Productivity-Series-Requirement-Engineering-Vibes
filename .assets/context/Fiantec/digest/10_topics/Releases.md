# Releases & Version Management

**Last Updated:** 2026-01-16
**Source:** `RAW/02_Change Management`

## Overview

This section tracks all FiANTEC releases, version history, and deployment certifications.

### Release Naming Convention
- Format: `R<YYYY>.<MM>` (e.g., R2024.07)
- Patches: `R<YYYY>.<MM>_Patch<YYYYMMDD>`

### Release Types
- **Major Release**: Significant new features (bi-monthly)
- **Minor Release**: Bug fixes and small enhancements
- **Patch/Hotfix**: Critical fixes between releases

### Certification Process
1. Development complete
2. Testing passed (all levels)
3. Business approval (Freigabezertifikat Business)
4. Technical approval (Freigabezertifikat Technical)
5. Production deployment authorization
